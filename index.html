DEFINITION

Was sind VPAs?

Ein virtueller persönlicher Assistent ist ein Software-Agent, der Aufgaben oder Dienste für eine Person auf der Grundlage von Befehlen oder Fragen ausführen kann, wie Z.B. To-Do-Listen, Wetter, Nachrichten usw., indem sie Spracherkennung, natürliche Sprachverarbeitung und maschinelles Lernen verwendet. Ein VPA kann auf einem Smartphone, Tablet, Computer oder einem intelligenten Lautsprecher installiert werden. Der Benutzer kann mit dem VPA sprechen und ihm Anweisungen geben, um Aufgaben wie das Senden von Textnachrichten, das Einrichten von Terminen, das Abspielen von Musik oder das Durchsuchen des Internets auszuführen. Der VPA kann auch personalisierte Empfehlungen basierend auf den Vorlieben des Benutzers geben. Die Funktionsweise eines VPA basiert auf der Verarbeitung von Spracheingaben durch den Benutzer, der Analyse der Anfrage durch die Software und der Ausgabe einer Antwort oder Aktion durch den VPA.
